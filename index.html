Project: Photo Gallery on Appengine (java)

A customer who wanted to have a photo album website that is easy to update. We decided to use G+ to upload and organize the photos. Once the photos are on G+ the website owner can login to an admin console and choose which albums are used on the site. The website uses these choices to create the site navigation.

Project: Pydoit a task tracking application (python/gtk)

Pydoit will provide you with reminders of the tasks or steps that you need to complete. You can mark them as completed for each day and review what days those tasks have been done. Each task is represented by a large button with a photo on it and optionally a sound. This application is intended to keep children on task but there is no reason adults could not use this system.
